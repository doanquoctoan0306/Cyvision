### What is this repository for? ###

Repo được tạo để lưu trữ source code cho ứng dụng di động chỉ đường cho người khiếm thị trên nền Android.

1- Hiển thị bản đồ
2- Lưu địa điểm hiện tại
3- Quản lý địa điểm
4- Định vị
5- Xác định hướng
6- Phát âm thanh
7- Chỉ đường
8- Huỷ chỉ đường
9- Quản lý số điện thoại khẩn
10- Gọi điện
11- Nhắn tin vị trí
12- Tìm đường
